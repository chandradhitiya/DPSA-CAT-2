Privacy-preserving ML dashboard

Compare standard ML (logistic regression + random forest baselines) against DPML in three places: noisy inputs (pre-processing), noisy gradients during training (DP-SGD / Opacus), and noisy predictions (post-processing).

What the UI shows after a run

  • BASELINE — clean data, no DP (utility reference).
  • PRE — Laplace vs Gaussian noise on features before training LR.
  • DP-SGD — Gaussian mechanism in optimizer (ε, δ budget via Opacus).
  • POST — Laplace vs Gaussian noise on probabilities from LR or RF.
